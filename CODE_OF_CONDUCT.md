# Contributor Covenant Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The full text is available at: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers at **conduct@opentabletop.org**.

All complaints will be reviewed and investigated promptly and fairly. The project team is obligated to maintain confidentiality with regard to the reporter of an incident.
